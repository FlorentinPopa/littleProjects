HTML (HyperText Markup Language)

  1. HTML Body
    • We put texts in paragraphs: <p> tag.
    • Inside a paragraph we can emphasize text with <em> or <strong>.
    • We can also use <span> (and give it some style in CSS).
    • Even if we put text in a few lines, it will be displayed in one line.
    • It will wrap only when not fitting its space.
    • To force line break we use <br /> tag.
    • Daca incepi un paragraf <p>……</p> atunci cand il inchizi trece automat pe randul urmator.
  2. STRUCTURE OF A DOCUMENT
    • Structure of a document
      • The basic unit of HTML document structure is a <div>.
      • A <div> (acronym for “division”) is used as container for other elements.
      • We can organize all the content inside a structure of divs.
      • It is then easier to apply style rules or JS scripts to the document.
      • <div id="header"> </div> --> id-ul ne ajuta sa dam un id unic unui div; pe care ulterior il putem folosi ca si cheie primara in CSS
      • Class= "…" --> clasa reprezinta un cumul
    • HTML Headings
      • Headings serve as titles or subtitles of the text.
      • HTML provides a range of headings to easily manage the structure of the text.
      • The main heading of a document is <h1>.
      • Then its subtitles are <h2>, and so on, down to <h6>.
    • HTML LISTED ELEMENTS
      • In the text we use lists to point out a number of items.
      • Unordered (bulleted) list is made with <ul> tag.
      • Ordered (numbered) list is made with <ol> tag.
      • Every list has to have elements inside.
      • To create a list element, we use <li> tags.
      • Lists can help us build the page structure (like menus).
    • HTML Tables
      • We use tables to show data.
      • We start a table with a <table> tag.
      • Table row: <tr> tag.
      • Table header cell: <th> tag.
      • Table cell: <td> tag.
    • HTML5 Document structure
      • The current version of HTML is HTML5.
      • When released, it came with many new tags.
      • Some of them can help us build a structure of a document.
      • We can use them instead of just divs everywhere.
      • New commands - sunt identice ca rol cu div-ul. Ele au ca scop sa fie usor de interpretate de SEO, samd
        • <article> - single entry (e.g. a blog post),
        • <section> - standalone section of a document,
        • <header> - contains top of the page or section, can contain some introductory content of a document,
        • <main>
        • <nav> - contains navigation links,
        • <footer> - footer of a document or of a section.
      • More elements:
        • <audio> - places an audio file on the website,
        • <video> - places a video file on the website,
        • <time> - places current time on the website,
        • <mark> - text highlight,
        • <dialog> - small dialog (popup) window.
    • HTML Comments
      • Comments can help developers get on with the structure of HTML files.
      • They can give some hint of what is what.
      • They are not visible in the browser.
      • However, they will be visible in the source code (or Developer Tools).
      • HTML Special characters
        • &nbsp; - non breaking space,
        • &laquo;, &raquo; - «, »
        • &gt;, &lt; - >, < (these can be useful, because < and > are needed to make HTML tags),
        • &euro; - €.
